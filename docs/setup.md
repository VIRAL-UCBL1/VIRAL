# Welcome to VIRAL

## Requirements

- Python 3.11
- [Ollama](https://ollama.com/).


## Installation

- Python requirements:

```bash
pip install -r requirements.txt
```

- [Ollama](https://ollama.com/download):

```bash
ollama run qwen2.5-coder
```

## Usage

```bash
cd src
python main.py
```

You can see help message by running:

```bash
python main.py -h
```
